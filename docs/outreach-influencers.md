## Outreach plan / approach

- **The ask**: keep it short, no "buy our token" energy. Explain $MEMES in one or two lines (treasury buys + redistributes pump.fun tokens to holders weekly, active from trade #1, community-voted via Realms, Fun IRL Hub coming) and ask for support, advice, or collaboration. A shoutout/RT is the realistic best case.
- **Channel**: X, public reply first if not mutuals, then DM (see Email reality check below, none of these have a public personal email except Helius's general inbox).
- **Sequencing**: don't send all 17 at once. Pilot batch first, picked for reach and thematic fit: Ansem (large pump.fun-native trading audience), Frank DeGods (IRL events track record), notthreadguy (meme culture reach). Use what lands with these to refine the message before the rest go out.
- **Angle (2026-06-15)**: lead with "you're one of the first we're reaching out to", offer a hand to all memes, the thesis that legit memecoins should unite and push for real IRL use (our project is partly built on that), and close with "any advice, help, or comment would mean a lot". Warmer/collaborative, not a "buy our token" pitch.
- **Note on the 2026-06-15 additions**: notthreadguy, CryptoGodJohn, SOLBigBrain, cozypront were added this round. Worth a quick check that each handle/account is still active before sending, CT accounts churn faster than this doc gets updated.
